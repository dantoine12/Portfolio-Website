Numbers To Words

The program convert numbers into word. The user will enter a number numer such 123,123 and the program will display one hundred twenty three thousand one hundred twenty three. The program was created written in Java on eclipse IDE. Writting this program be easy once i figured out that if I can decode the three number then I should be able to decode every other number, regardless of the decimal point. I first had to decode the ones place, then the tens, and the hundreds. Once I was able to fo that, it didn't matter if the number was 6,000,000 converting it into words would be the same process.
