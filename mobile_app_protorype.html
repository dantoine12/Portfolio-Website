Mobile App Prototype

This program was written in C# using windows Form. The purpose of this app was to demostrate how a typical CRUD application would work. The program allows user to enter a license plate and if that licnese plate is in the database then the program will return the name and phone number of the vehicle owner. The program also allows the user to enter a new person into the database. When I writing this program I had to learn how to work with a sql database, which is something I've never done before. I also had to learn how to use sql statements as stored procedures.
